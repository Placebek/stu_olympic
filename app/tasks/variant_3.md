## 1. Жойғаннан кейінгі максималды бірегей ішкі массив қосындысы (Оңай)
### Шарт 

`nums` бүтін сандар массиві берілген.

Сізге `nums`-тен кез келген элементтерді жоюға рұқсат етілген, бірақ массив бос болмауы керек. Жойғаннан кейін `nums`-тің ішкі массивін таңдаңыз, мұндай:

1. Ішкі массивтің барлық элементтері **бірегей** (қайталанбайды).
2. Ішкі массив элементтерінің **қосындысы барынша үлкен**.

Осындай ішкі массивтің максималды қосындысын қайтарыңыз.

**Мысалдар:**

**1-мысал:**
- Кіріс: `nums = [1,2,3,4,5]`
- Шығыс: `15`
- Түсінік: Ешнәрсе жоймай бүкіл массивті таңдаймыз — максималды қосынды.

**2-мысал:**
- Кіріс: `nums = [1,1,0,1,1]`
- Шығыс: `1`
- Түсінік: `nums[0]=1`, `nums[1]=1`, `nums[2]=0`, `nums[3]=1` жоямыз. Қалған `[1]` ішкі массивін таңдаймыз.

**3-мысал:**
- Кіріс: `nums = [1,2,-1,-2,1,0,-1]`
- Шығыс: `3`
- Түсінік: `nums[2]=-1` және `nums[3]=-2` жоямыз. `[1,2,1,0,-1]`-ден `[2,1]` ішкі массивін таңдаймыз, қосынды = `3`.

## Русский: Максимальная сумма уникального подмассива после удаления (Лёгкий)
### Условие 

Дан массив целых чисел `nums`.

Разрешается удалить любое количество элементов из `nums`, не делая его пустым. После удалений выберите подмассив `nums` такой, что:

1. Все элементы подмассива **уникальны** (не повторяются).
2. **Сумма элементов** подмассива максимальна.

Верните максимальную сумму такого подмассива.

**Примеры:**

**Пример 1:**
- Ввод: `nums = [1,2,3,4,5]`
- Вывод: `15`
- Пояснение: Выбираем весь массив без удалений — максимальная сумма.

**Пример 2:**
- Ввод: `nums = [1,1,0,1,1]`
- Вывод: `1`
- Пояснение: Удаляем `nums[0]=1`, `nums[1]=1`, `nums[2]=0`, `nums[3]=1`. Выбираем подмассив `[1]`.

**Пример 3:**
- Ввод: `nums = [1,2,-1,-2,1,0,-1]`
- Вывод: `3`
- Пояснение: Удаляем `nums[2]=-1` и `nums[3]=-2`. Из `[1,2,1,0,-1]` выбираем подмассив `[2,1]`, сумма = `3`.

## 2. Теңдестірілген K-факторлы ыдырау (Орташа)
### Шарт 

Екі бүтін сан `n` және `k` берілген.

`n` санын дәл `k` оң бүтін санға бөліңіз, олардың **көбейтіндісі `n`-ге тең** болатындай.

Кез келген екі санның **айырымы минималды** болатын бөлуді қайтарыңыз. Нәтижені кез келген ретпен беруге болады.

**Мысалдар:**

**1-мысал:**
- Кіріс: `n = 100, k = 2`
- Шығыс: `[10, 10]`
- Түсінік: `10 × 10 = 100`, максималды айырым = `0` — минималды мүмкін.

**2-мысал:**
- Кіріс: `n = 44, k = 3`
- Шығыс: `[2, 2, 11]`
- Түсінік:
  - `[1, 1, 44]` → айырым 43
  - `[1, 2, 22]` → айырым 21
  - `[1, 4, 11]` → айырым 10
  - `[2, 2, 11]` → айырым **9** (ең кіші)

**Шектеулер:**
- `4 <= n <= 10^5`
- `2 <= k <= 5`
- `k` саны `n`-нің оң бөлгіштерінің жалпы санынан қатаң аз.

## Русский: Сбалансированное K-факторное разложение (Средний)
### Условие 

Даны два целых числа `n` и `k`.

Разбейте `n` ровно на `k` положительных целых чисел так, чтобы их **произведение было равно `n`**.

Верните любое разбиение, в котором **максимальная разница** между любыми двумя числами минимальна. Результат можно вернуть в любом порядке.

**Примеры:**

**Пример 1:**
- Ввод: `n = 100, k = 2`
- Вывод: `[10, 10]`
- Пояснение: `10 × 10 = 100`, разница = `0` — минимально возможная.

**Пример 2:**
- Ввод: `n = 44, k = 3`
- Вывод: `[2, 2, 11]`
- Пояснение:
  - `[1, 1, 44]` → разница 43
  - `[1, 2, 22]` → разница 21
  - `[1, 4, 11]` → разница 10
  - `[2, 2, 11]` → разница **9** (наименьшая)

**Ограничения:**
- `4 <= n <= 10^5`
- `2 <= k <= 5`
- `k` строго меньше общего числа положительных делителей `n`.

## 3. Келесі ерекше палиндром саны (Қиын)
### Шарт 

`n` бүтін саны берілген.

Сан **ерекше** деп аталады, егер:
- Ол **палиндром** болса (солдан оңға және оңнан солға оқығанда бірдей).
- Сандағы әр `k` цифры дәл `k` рет кездессе.

`n`-нен **қатаң үлкен** ең кіші ерекше санды қайтарыңыз.

**Мысалдар:**

**1-мысал:**
- Кіріс: `n = 2`
- Шығыс: `22`
- Түсінік: `22` — палиндром, `2` цифры дәл 2 рет кездеседі. 2-ден үлкен ең кіші ерекше сан.

**2-мысал:**
- Кіріс: `n = 33`
- Шығыс: `212`
- Түсінік: `212` — палиндром, `1` цифры 1 рет, `2` цифры 2 рет кездеседі. 33-тен үлкен ең кіші ерекше сан.

**Шектеулер:**
- `0 <= n <= 10^15`

## Русский: Следующее особое палиндромное число (Сложный)
### Условие 

Дано целое число `n`.

Число называется **особым**, если:
- Оно является **палиндромом** (читается одинаково слева направо и справа налево).
- Каждая цифра `k` в числе встречается ровно `k` раз.

Верните наименьшее особое число, строго большее `n`.

**Примеры:**

**Пример 1:**
- Ввод: `n = 2`
- Вывод: `22`
- Пояснение: `22` — палиндром, цифра `2` встречается ровно 2 раза. Наименьшее особое число, большее 2.

**Пример 2:**
- Ввод: `n = 33`
- Вывод: `212`
- Пояснение: `212` — палиндром, цифра `1` встречается 1 раз, цифра `2` — ровно 2 раза. Наименьшее особое число, большее 33.

**Ограничения:**
- `0 <= n <= 10^15`
