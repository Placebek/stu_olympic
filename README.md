# Maximum Subarray With Equal Products

## Задание

Дан массив положительных целых чисел `nums`.

Подмассив `arr` называется **product-equivalent**, если выполняется условие:

где:

- `prod(arr)` — произведение всех элементов подмассива  
- `gcd(arr)` — наибольший общий делитель всех элементов  
- `lcm(arr)` — наименьшее общее кратное всех элементов  

Подмассив — это **непустая последовательность элементов массива, расположенных подряд**.

Необходимо найти **длину самого длинного подмассива**, который удовлетворяет этому условию.

---

## Примеры

### Пример 1

**Input**
nums = [1,2,1,2,1,1,1]

**Output**
5

**Explanation**

Самый длинный подходящий подмассив:
[1,2,1,1,1]

Проверка условия:
prod = 2
gcd = 1
lcm = 2
prod = gcd × lcm
2 = 1 × 2

---

### Пример 2

**Input**
nums = [2,3,4,5,6]

**Output**
3

**Explanation**

Подходит подмассив:
[3,4,5]
